SumType.opEquals

sumtypeSumType

Compares two SumTypes for equality.

Two SumTypes are equal if they contain values of the same type, and those values are equal.

struct SumType(TypeArgs...)
bool

TypeArgs.length > 0
&&
TypeArgs.length < size_t.max
)

Bugs

Comparison of SumTypes that contain a const(This)[], either directly or indirectly, will sometimes give the wrong result by comparing the arrays for reference equality (is) instead of value equality (==).
